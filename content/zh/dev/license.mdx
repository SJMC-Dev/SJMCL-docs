# 开源协议

该软件依据 [GNU 通用公共许可证第 3 版（GPLv3）](https://github.com/UNIkeEN/SJMCL/blob/main/LICENSE) 进行分发。

根据 GPLv3 许可协议第 7 条的规定，我们要求您在分发本软件的修改版本时，除了遵守 GPLv3 协议外，还必须遵守以下 [附加条款](https://github.com/UNIkeEN/SJMCL/blob/main/LICENSE.EXTRA)：

1. 使用与 SJMCL 或 SJMC Launcher 不同的软件名称；
2. 不得移除向我们的统计服务器发送用户统计信息的功能（位于 `src-tauri/src/utils/sys_info.rs`），并且必须仅发送带有前缀（前缀需为两个以上字母，例如 `XX-0.0.1`）的版本号；
3. 在您的代码仓库 README 文件、分发网站或发布帖、支持文档以及软件的“关于”页面中，明确标注您的程序基于 SJMCL，并提供原始仓库的链接。
